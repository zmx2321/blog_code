Ghelper 版本从v2.8.0 起的版本是基于浏览器最新的内核架构MV3开发的，MV3内核是Chrome系列浏览器的最新内核，他的特点是底资源占用，底内存占用，异步事件处理，是浏览器未来发展的方向。

v2.7.6 是基于老的内核MV2架构开发的，为了照顾一些老式电脑系统的用户,如果您的电脑能安装的最新的浏览器版本无法使用v2.8.0以后的版本,可以试试这个旧的版本安装。

强列推荐用户安装最新的版本，最新的版本目前已经非常稳定.占用资源非常少。

Chrome / Edge / 360 都可以用.
在打浏览器，在地址栏最右边有三个点点开它，展开菜单.
选择 "更多工具“ -> 扩展程序
打开"扩展程序“ 在这里能看到所有已安装的扩展程序。
将你下载Ghelper 压缩包中 Ghelper-v2.8.1.crx 文件拖到这个扩展程序页面上，就能完成安装，如果拖过去没反应。你需要打开开发者模式。
Chrome 在"扩展程序"的右上角，有个“开发者模式”打开它，再将CRX拖进来就行了。
Edge 浏览器在"扩展“程序的左边菜单中有一个”开发人员模式“ 打开它，再将CRX文件拖进来就行了。

其它Chrome内核的浏览器安装方法差不多。
